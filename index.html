Problem while loading

Due to changes on how the picrew backup tool archives files, sometimes enabling compatibility mode is necessary

To enable compatibility mode add &compat-mode=old1 or &compat-mode=NO_NAMES_IN_PATH to the end of the url

No ID specified

Backups should be placed in the data directory

To load a backup, add ?id=<folder name> to the end of the url and replace <folder name> with the id of the backup you placed in the data folder
